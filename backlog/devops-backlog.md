# IC63 Migration Acceleration Backlog

## EPIC 1 - Unified Prototype Experience
### Feature 1.1 - Single Entry Point
- Story: As an SE, I can choose Readiness, Intake, or Both from one interface.
- Story: As a lead, I can track which option is used per account.

### Feature 1.2 - Shared Data Model
- Story: As a contributor, I can reuse a common YAML schema across modules.
- Story: As a reviewer, I can trace generated outputs to normalized fields.

## EPIC 2 - Readiness Assessment Module
### Feature 2.1 - Guided Assessment
- Story: As an SE, I answer a structured domain-based question set.
- Story: As an SE, I can mark unknowns without blocking progress.

### Feature 2.2 - Readiness Scoring
- Story: As a lead, I get a commit-confidence recommendation and rationale for stage movement.
- Story: As a partner, I can see blockers with owner and due dates.

### Feature 2.3 - Dependency Mapping
- Story: As CSU/Factory, I can see sequence dependencies before nomination.

## EPIC 3 - Factory Intake Module
### Feature 3.1 - Note Ingestion
- Story: As an SE, I can paste/upload discovery notes for transformation.

### Feature 3.2 - Nomination Pack Generation
- Story: As CSU/Factory, I receive a structured pack with assumptions, risks, and open questions.

### Feature 3.3 - Quality Guardrails
- Story: As a reviewer, I can detect missing decisions and low-confidence assumptions.

## EPIC 4 - Feedback and Continuous Improvement
### Feature 4.1 - Lightweight Feedback Capture
- Story: As an SE, I can submit feedback in under 1 minute.

### Feature 4.2 - Iteration Review Dashboard
- Story: As IC63 owners, we can identify top recurring friction points per cycle.

### Feature 4.3 - Release Cadence
- Story: As a team, we publish vNext every 3-4 weeks with high-value fixes only.

## Initial Priority (Now / Next / Later)
- Now: EPIC 1, EPIC 2 Feature 2.1/2.2, EPIC 3 Feature 3.2, EPIC 4 Feature 4.1
- Next: EPIC 2 Feature 2.3, EPIC 3 Feature 3.3, EPIC 4 Feature 4.2
- Later: deeper analytics, account-level benchmarking, expanded automation
