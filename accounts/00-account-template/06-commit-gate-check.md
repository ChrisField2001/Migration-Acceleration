# 06 - MCEM Commit Gate Check (SE)

## Purpose
Use this checklist to confirm whether the opportunity has enough confidence and data to move from Uncommitted to Committed.

## Minimum Inputs (All Required)
- [ ] Business outcome and deadline are clear.
- [ ] Economic buyer and approval path are known.
- [ ] Wave-1 scope and out-of-scope are explicit at presales level.
- [ ] Readiness snapshot exists for identity, network, subscription, and dependencies.
- [ ] Top 3 risks include impact, owner, and target date.
- [ ] Unknowns that affect commitment have owner and date to resolve.
- [ ] 2-week customer and Microsoft action plan is defined.

## Commit Artifacts Check
- [ ] Scope note prepared.
- [ ] ROM estimate prepared.
- [ ] High-level plan prepared.
- [ ] Risk log prepared.
