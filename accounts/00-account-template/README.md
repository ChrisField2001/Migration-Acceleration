# Account Working Folder Template

Use this folder as the starting point for each new customer account.

## How To Use
1. Copy this folder and rename it to your account name, for example: `accounts/NHS-Trust-A`.
2. Fill files in order from `01` to `05`.
3. Use `06-commit-gate-check.md` to confirm Uncommitted -> Committed readiness.
4. Share `02` and `03` with CSU/Factory.
5. Log quick feedback in `04`.

## File Order
- `01-discovery-answers.md`: Capture minimum commit-critical inputs (presales level only).
- `02-readiness-output.md`: Readiness summary, blockers, and dependencies.
- `03-factory-nomination-pack.md`: Handoff pack for CSU/Factory.
- `04-feedback.md`: 1-minute usability feedback.
- `05-engagement-log.md`: Timeline, decisions, and review notes.
- `06-commit-gate-check.md`: Gate checklist to support stage movement recommendation.

## Definition Of Done For One Account
- Opportunity can be recommended for Uncommitted -> Committed decision.
- Wave-1 scope is explicit.
- Top blockers have named owner and target date.
- Dependency sequence is clear.
- Nomination pack has been sent for review.
- Feedback has been captured.
